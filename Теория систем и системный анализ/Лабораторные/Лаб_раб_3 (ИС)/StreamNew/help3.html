Как построить модель AnyLogic

При построении модели в формате AnyLogic программа проверяет входные данные на корректность. Затем производится анализ модели, устранение противоречий путем ввода дополнительных элементов, построение системы уравнений, описывающей модель, представление модели в формате AnyLogic (в виде объекта).

1. Проверка правильности модели

Перед построением модели для AnyLogic можно проверить модель на наличие ошибок. На этапе проверки определяются все возможные ошибки, которые могут встретиться при построении модели: ошибки ввода или логические ошибки модели.

Для проверки правильности модели в меню выберите Модель->Проверить (или нажмите клавишу F6). При успешном завершении проверки выдается соответствующее сообщение. Если при проверке обнаружены ошибки, программа выдает сообщения с их описанием.

2. Построение модели для AnyLogic

Для того, чтобы построить файл AnyLogic с моделью схемы, выберите Модель->Построить (или нажмите клавишу F5). При построении модели программа предложит выбрать файл, где она будет создана. Можно выбрать уже существующий файл AnyLogic или создать новый.

При выборе существующего файла имеется возможность добавить модель к уже имеющимся в файле объектам или переписать файл, поместив в него только создаваемую модель.

Если модель создана успешно, программой выдается сообщение об этом. Если в процессе построения модели обнаружены ошибки, появляются сообщения с их описанием.
